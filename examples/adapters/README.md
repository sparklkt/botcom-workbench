# Example operating adapters

Copy these JSON files into your local adapters folder:

```bash
mkdir -p ~/BotCom/adapters
cp examples/adapters/*.json ~/BotCom/adapters/
```

Then open **BotCom OS**. The one-person company operating map will use the adapter signals to update the customer, revenue, and delivery modules.

Do not store API keys, passwords, tokens, customer secrets, or private account exports in adapter files. Keep adapter JSON small and summary-level.
